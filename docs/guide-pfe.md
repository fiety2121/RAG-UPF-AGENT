# Guide PFE — Projet de Fin d'Etudes

## Calendrier PFE
- Choix du sujet : avant le 15 novembre
- Validation encadrant : avant le 1er decembre
- Depot rapport final : avant le 1er juin
- Soutenances : juin - juillet

## Procedure
1. Consulter la liste des sujets sur le portail
2. Contacter l'enseignant encadrant
3. Faire valider le sujet par la coordination
4. Rediger le rapport selon le modele officiel
5. Deposer 3 exemplaires relies a la scolarite

## Criteres d'evaluation
- Rapport ecrit : 40%
- Soutenance orale : 40%
- Suivi encadrant : 20%
